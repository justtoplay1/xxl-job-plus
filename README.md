## 介绍

xxl-job-plus是xxl-job的增强包，提供注册中心能力，支持监测nacos服务上下线，executor自动启动、刷新、停止。完美兼容xxl-job-admin，不再需要为executor单独配置ip和端口，也不需要配置admin地址，自动从注册中心获取
